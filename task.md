# 更好的物质聚合器 - Better Molecular Assembler

## 1. 红石兼容方案
### 功能
- **红石信号控制自动导出**
- **可反转信号**（像 ME 触发总线那样）

### 模式
1. **自动导出开启**
   合成完成 → 直接输出到 ME 网络
2. **自动导出关闭**
   产物留在内部，不弹出（原始行为）

### 红石逻辑
- 模式：**忽略 / 正常 / 反转**
  - **忽略**：不响应红石信号（原始行为，默认）
  - 正常：有红石信号 → **自动导出**
  - 反转：有红石信号 → **不导出**

## 2. 选取聚合器逻辑
同一 ME 网络内选择目标聚合器：
1. **优先级高优先**
2. 同优先级 → **后放置的先选**

物品**只发给一台**，不串网、不吞物品、不冲突。

---

## 3. 完整流程
无线终端垃圾桶丢物品 → 同 ME 网络按规则选一台物质聚合器 → 物品入内 → 合成 → 根据红石+开关自动导出/保留。

# 📘 GuideMe 结构（你直接复制做文档）
# Better Molecular Assembler
更好的物质聚合器

## 概述
- 基于 AE2 物质聚合器的增强版
- 接入 ME 网络，不串网
- 无线终端垃圾桶自动将物品输入本机
- 支持多台共存，按规则分配输入

## 功能
1. **ME 网络绑定**
   仅响应同网络无线终端的垃圾桶输入。

2. **优先级排序**
   - 优先级高的机器优先接收物品
   - 同优先级：后放置的机器优先

3. **自动导出（可红石控制）**
   - 开启：产物自动存入 ME 网络
   - 关闭：产物保留在内部
   - 支持红石信号控制
   - 支持红石信号反转

4. **无线终端垃圾桶联动**
   丢弃物品 = 输入同网络最高优先级的更好的物质聚合器。

## 物品输入规则
- 无线终端垃圾桶 → 同网络
- 排序：优先级（高→低） → 放置时间（新→旧）
- 只选择**一台**机器输入

## 红石说明
- 正常模式：红石信号 → 自动导出
- 反转模式：红石信号 → 停止导出
