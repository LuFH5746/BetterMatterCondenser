# Better Matter Condenser

This is a placeholder guide page for **Better Matter Condenser**.

## Features

- Intercept items trashed from AE2 wireless terminals into Matter Condensers.
- Supports AE2WTLib standalone wireless terminal trash slots.
- Select receiving condenser by priority or last placed order.
- Redstone-controlled auto-export of condenser products into the ME network.
- Configurable per-tick export rate limit.
